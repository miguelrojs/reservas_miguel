# Reservas Miguel
Proyecto de sistema de reservas con frontend y backend.
## 🚀 Requisitos
- Node.js
- npm
---
## 📦 Instalación
Clonar el repositorio:
git clone https://github.com/miguelrojs/reservas_miguel.git
cd reservas_miguel
---
## 🔧 Backend
cd backend
npm install
npm run start:dev
---
## 💻 Frontend
cd frontend
npm install
npm start
---
## 🌱 Seed (datos iniciales)
cd backend
npm run seed
---
## 📌 Notas
- Asegurarse de configurar el archivo .env en el backend.
- El backend corre normalmente en http://localhost:3000
- El frontend en http://localhost:4200
